# Evening Anchor — 晚间锚点生成器

## 1. Concept & Vision

**"晚间节拍器"**：在一天与夜晚之间建一个温柔的缓冲区。它不是待办清单，不是日程表——它是一个**只给你一条路走**的向导。核心体验是：用户花1分钟填信息，系统立即输出"照着做"的时间轴，消除所有后续决策。

界面感受：**傍晚窗边的笔记本**——温暖的纸质感，柔和的灯光色，手写感的标题，但底下是精确的时间切割。

## 2. Design Language

### Aesthetic Direction
**Warm Twilight Minimalism** — 受日式极简和欧洲独立杂志排版启发。不是冷冰冰的工具感，而是"为自己认真生活的人准备的私人指南"。

### Color Palette
```

--bg-card:       #1A1830   /* 卡片底色 */

```

### Typography
- **Display**: `Cormorant Garamond` (600/700) — 有温度的衬线体，标题用
- **Body**: `Noto Sans SC` (400/500) — 中文友好，清晰可读
- **Mono**: `JetBrains Mono` (400) — 时间数字用，等宽精确感

### Spatial System
- 基础单位：8px
- 卡片圆角：16px
- 大量留白，内容区宽度 max 560px（单列，聚焦）
- 垂直节奏：区块间 32-48px

### Motion Philosophy
- **Page load**: 表单区淡入 + 上浮（opacity 0→1, translateY 20px→0, 600ms ease-out）
- **生成时**: 计划卡片依次出现（staggered, 150ms 间隔）
- **时间轴**: 每个 block 从左滑入（translateX -20px→0）
- **切换模式**: 主计划/保底版本 淡入淡出（300ms crossfade）

## 4. Features & Interactions

### 核心功能

**① 表单输入**
- 到家时间：原生 time input，简化（不带秒）
- 精力/场景/目标：自定义按钮组，点击即选中，高亮样式
- 明日事项：有按钮 + 条件显示时间 input

**② 计划生成算法**
输入 → 生成时间轴，每块包含：
- 时间范围（开始时间 + 结束时间）
- 动作名称
- 时长（分钟）
- 强度标签（轻松 / 适度 / 专注）
- 图标
- 说明文字

算法规则：
- 到家后第一件事永远是最小阻力动作（换衣服/收包/喝水，<5分钟）
- 高精力：主目标任务安排在精力黄金期（到家后1-2小时）
- 中精力：主目标任务缩短+拆碎
- 低精力：只保留核心生存+放松，主目标降级或删除
- 有明日事项：睡前留出"准备"块（整理包/定闹钟）
- 有娃：任务块与亲子时间交叉编排
- 独居高精力：可安排1-2个并行任务（边做饭边听播客）

**③ 保底版本逻辑**
- 触发：精力=低 或 家庭场景=有娃+精力=中
- 特点：块数减少（3-4块），每块时长缩短，总时长压到1.5-2小时
- 保留：晚餐 + 一个放松活动 + 睡前仪式
- 原则：不做任何需要意志力的事

**④ 交互细节**
- 表单提交：按钮显示 loading 状态（2秒模拟生成延迟）
- 计划生成后：表单淡出 → 计划淡入
- 点击"重新填写"：计划淡出 → 表单淡入，数据保留
- 点击"查看保底版本"：展开折叠区
- 第一步卡片：有特殊的边框高亮 + 脉冲动画

## 5. Component Inventory

### EnergyButton
- 状态：default（深底+边框）/ selected（琥珀色填充+白字）/ hover（边框变亮）
- 三个：低⚡🟡 / 中⚡🟠 / 高⚡🔴

### SceneButton / GoalButton
- 同 EnergyButton，但图标+文字组合
- 选中时：accent-cool 填充

### TimeInput
- 深色 input，focus 时边框变琥珀色
- 显示时钟图标

### PlanBlock
- 左侧竖线（彩色，根据强度变化）
- 时间（mono字体）+ 动作名（衬线体）
- 元信息行：时长 · 强度 · 说明
- 第一个块：加粗边框 + 琥珀色背景淡色 + "▶" 标记

### LowPowerSection
- 默认折叠（显示触发文字"🔋 精力不足时的保底方案"）
- 展开后：更简洁的块样式，文字偏灰

### GenerateButton
- 全宽，琥珀色渐变背景
- Hover：亮度提升 + 轻微上移
- Loading：显示旋转图标 + "生成中..."
- Disabled：灰色，不可点击

## 6. Technical Approach

- **单文件 HTML**（内嵌 CSS + JS，无依赖）
- **字体**：Google Fonts CDN（Cormorant Garamond + Noto Sans SC + JetBrains Mono）
- **无框架**：纯原生 JS，算法硬编码
- **状态管理**：简单对象 + DOM 更新
- **动画**：CSS transitions + keyframes
- **数据存储**：无（每次刷新重填）
